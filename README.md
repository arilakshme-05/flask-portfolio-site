# Flask Portfolio Website
##Project Description
The goal is to build a personal portfolio website using Flak with pages for:
-Home Page
-Contact Form
-Simple Styling with CSS
##Tech Stack
-Python
-Flask
-HTML
-CSS
##How to Run
1.Install Flask
pip install flask
2.Run the app
python app.py
3.Open in browser
http://127.0.0.1:5000/
##Features
Flask routing
Template rndering with Jinja2
Contact form with POST method
Static file handling.
